Quick(er) Access

For easier access in the future to the world of Anyarium, you can install the web app:

iOS: Tap Share, then Add to Home Screen
Android: Tap Install in the Automatic prompt (convenient!)
Edge: Click "App availible" in the toolbar, then "Install"
Chrome: Click install in the Install App? Prompt OR ⋮ > Cast, save and share > Install site as app
